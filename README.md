# IR Remote Control Utilities

Arduino utilities for Infrared (IR) Remote Control protocols and codes.

&nbsp;

This project use the following electronic components:
- 1 x Arduino Nano v3.0
- 1 x Breadboard
- 1 x IR LED Emiter
- 1 x IR LED Receiver
- 1 x 220 ohm resistor
- wires
